# 문제집 파이썬/자바 코드 안내

강의와 문제집에 수록된 문제들에 대한 파이썬/자바 코드 요청이 꾸준히 있어 왔지만, 여러 가지 사정으로 현상 유지에도 벅차 제공하지 못하고 있었습니다.

그러던 중 [hanXen](https://github.com/hanXen)님께서 0x11강까지의 모든 문제에 대해 퀄리티 높은 파이썬 코드들을 공개해 주셔서, 해당 리포지토리를 소개드립니다 
([리포지토리 링크](https://github.com/hanXen/basic-algo-lecture-python)). 학습에 도움이 되시길 바랍니다.

또한 개발자 지인의 도움을 받아 LLM을 이용해 저와 여러분이 함께 작성해주신 C++ 정답 코드를 파이썬과 자바로 포팅하는 방식으로 남은 문제들에 대한 파이썬 및 자바 정답 코드를 생성하려고 합니다. 확답을 드릴 수는 없지만 2월 안에 완성할 수 있도록 노력해 보겠습니다.
